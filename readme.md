### 1. Clone
```
git clone https://github.com/akmaladisa/backend-tahap-2.git
```

### 2. Install packages
```
cd backend-tahap-2
```
```
npm install
```

### 3. Database set up
```
DB name: backend-tahap-2
DBMS: MySQL
```

### 4. Sequelize CLI set up
```
npm install sequelize-cli
```

### 5. DB Migrate
```
sequelize db:migrate 
```

### 6. Start server
```
npm run dev
```

### 7. URL
```
http://localhost:8888/
```
